## Cómo abrir en Google Chrome

- Haz doble clic en `index.html`.
- O haz clic derecho en `index.html` y elige "Abrir con" -> "Google Chrome".

## Usando un servidor local (opcional)

Si tienes Python instalado, puedes ejecutar un servidor local desde la terminal:

```powershell
cd "c:\Users\William\OneDrive\Escritorio\AssetsWeb"
python -m http.server 5500
```

Luego abre `http://localhost:5500` en Chrome.
